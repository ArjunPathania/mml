# PCA From Scratch

## Overview
This project implements Principal Component Analysis (PCA) from scratch using NumPy and validates results using sklearn.

## Features
- Manual PCA implementation
- Synthetic dataset generation
- Visualization of projections and reconstruction
- Comparison with sklearn PCA

## Concepts Covered
- Mean centering
- Covariance matrix
- Eigen decomposition
- Dimensionality reduction
- Reconstruction

## How to Run
pip install -r requirements.txt
python main.py
